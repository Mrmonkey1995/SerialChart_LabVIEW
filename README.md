# 基于LabVIEW的串口示波器

* 用途：<br>方便把串口的数据传到电脑上实现绘图功能方便分析数据

* 串口数据格式：<br>C1:123.0\r\nC2:200.1\r\nC3:8.89\r\n<br>类似即可
* 说明：
  三个通道不需要同时使用。
